use wordpress bulitin video/audio player in widgets

== Description ==

since wordpress has the ability to show and play videos and audios by shortcodes,why not use these in widget mode?

all available media shortcodes settings are available in this plugin also.

Features:

*   Simple!
*   Support Audio and Video
*   Use wordpress bulitin players
*   no limits in widgets
